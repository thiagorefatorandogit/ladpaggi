
🚀 FocusApp — Landing Page de Produtividade

Landing page desenvolvida para simular a página de apresentação de um aplicativo de produtividade chamado FocusApp.

O objetivo do projeto foi praticar HTML, CSS e JavaScript puro, criando uma página moderna, interativa e com funcionalidades reais de interface.

🖥️ Preview do Projeto

Landing page com:

Hero section com call-to-action
Seção de benefícios
Depoimentos de usuários
Seção de download
Popup de captura de email
Dark Mode 🌙

✔️ Landing page completa de produto
✔️ Alternância de Tema Claro/Escuro (Dark Mode)
